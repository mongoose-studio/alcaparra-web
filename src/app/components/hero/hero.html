El lenguaje para {{ currentPhrase }}

AlcaparraLang es un lenguaje embebible de propósito específico para lógica de negocio determinista. Sin acceso a I/O, sin estado global, sin efectos secundarios: un script recibe un contexto de datos, ejecuta un grafo de transformaciones puras y emite un resultado JSON estructurado. La inmutabilidad es explícita (let vs var), los imports son acíclicos y el runtime garantiza idempotencia por construcción — no por convención.

/// Calcula el bono mensual.
/// @param base   Sueldo base
/// @param factor Porcentaje (0–1)
fn calcular_bono(base, factor) {
    let tope = 500_000;
    var resultado = base * factor;
    if resultado > tope {
        resultado = tope;
    }
    emit resultado;
}

main {
    let bono = calcular_bono(SUELDO_BASE, 0.15);
    emit { bono: bono };
}
