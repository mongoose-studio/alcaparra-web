Instalación manual del LSP

Si prefieres instalar solo el servidor LSP sin el runtime completo:

  1. Descarga el binario alcaparra-lsp para tu plataforma desde GitHub Releases
  2. Colócalo en tu PATH (ej. /usr/local/bin/alcaparra-lsp) y dale permisos de ejecución: chmod +x alcaparra-lsp
  3. Configura la ruta en el plugin de tu IDE si difiere del valor por defecto
